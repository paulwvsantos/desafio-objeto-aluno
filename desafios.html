Desafios de Objetos

Para quem está começando do zero · você resolve no desafios.js

Como usar

Passo a passo

  1. Aperte F12 e abra a aba Console.
  2. Abra o arquivo desafios.js no editor.
  3. Cada tarefa indica o tipo (VEJA E PREVEJA, MODIFIQUE ou CRIE) e o resultado esperado em comentário.
  4. Escreva sua resposta no espaço marcado, salve e aperte F5.

Regra que salva: um objeto agrupa dados que andam juntos. Cada dado tem uma etiqueta e um valor. Para ler: objeto.propriedade

Os três tipos de tarefa

Como cada tarefa funciona

  1. VEJA E PREVEJA — o código já está pronto. Antes de rodar, escreva o que você acha que vai aparecer. Depois rode e confira.
  2. MODIFIQUE — mude um pedacinho do código que já existe.
  3. CRIE — escreva você mesmo, do zero.

A lista sobe em níveis: criar e ler → alterar e adicionar → objeto + função → fábrica de objetos. Cada tarefa traz o resultado esperado para você se autocorrigir. Terminou? Veja o panorama no desafio_aluno.md.
